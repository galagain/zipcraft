ZipCraft Modrinth Pack Builder

Avant de commencer :

  1. Téléchargez et installez l’installateur Fabric pour la version de Minecraft voulue.
  2. Relancez le launcher Minecraft et sélectionnez le profil Fabric correspondant.
  3. Ici, générez le ZIP de mods puis placez son contenu dans %APPDATA%\.minecraft\mods (sous Windows, %APPDATA% pointe sur Roaming).

Tests unitaires rapides

Cliquez sur Exécuter les tests pour vérifier les fonctions clés sans appeler l’API.

(en attente)

Une fois le ZIP téléchargé, placez manuellement les fichiers dans le dossier %APPDATA%\\.minecraft\\mods (sous Windows, %APPDATA% ouvre directement Roaming). Sous macOS, utilisez ~/Library/Application Support/minecraft/mods, et sous Linux ~/.minecraft/mods.

ZipCraft — utilise l’API publique de Modrinth (api.modrinth.com/v2).
